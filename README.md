GeckoAI Kalshi

GeckoAI Kalshi is an automated Kalshi trading bot focused on BTC 15-minute prediction markets. It monitors live market data, identifies qualifying entries, manages position sizing, and records every trade for analytics and future machine learning improvements.

Features
📡 Live WebSocket price streaming
⚡ Automated trade execution
💰 Dynamic balance-based position sizing
📊 Session statistics dashboard
📝 Detailed trade logging
☁️ Optional global trade analytics
🔔 Discord notifications
🖥️ Windows, macOS & Linux support
🚀 One-click startup scripts
⚙️ First-run configuration wizard
Installation
git clone https://github.com/MaskedMajic/GeckoAi-Kalshi.git
cd GeckoAi-Kalshi
pip install -r requirements.txt

Run the setup wizard:

python start.py

Then launch:

Windows

Run.bat

macOS / Linux

./Mac\ &\ Linux\ Start.sh
